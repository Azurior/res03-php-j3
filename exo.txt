La liste de tous les utilisateurs => 
SELECT * FROM users

La liste de tous les utilisateurs rangée par noms de famille => 
SELECT * FROM users ORDER BY last_name ASC

Le dernier utilisateur inscrit => 
SELECT id FROM users ORDER BY registration_date DESC

La liste de tous les utilisateurs fêtant leur anniversaire ce mois-ci => 
SELECT * FROM users WHERE MONTH(birthdate) = MONTH(CURRENT_TIMESTAMP)

Le nombre total d'utilisateurs => 
SELECT COUNT(id) FROM users 

La liste de tous les utilisateurs associés à leurs villes respectives => 
SELECT users.*, city FROM users JOIN addresses ON user_id = id

La liste de tous les utilisateurs vivant à une adresse sans numéro => 
SELECT users.*, city FROM users JOIN addresses ON user_id = id WHERE number IS NULL

La liste de tous les produits dont le prix est supérieur à 1 000 € =>
SELECT * FROM products WHERE price > 1000

La liste de tous les produits associés à leurs photos respectives =>
SELECT products.*, pictures FROM products INNER JOIN caption ON products_id = id 

La liste de tous les produits appartenant à la catégorie « Voyage » =>
SELECT * FROM products JOIN products_categories ON products_categories.product_id = products.id JOIN categories
ON categories.id = products_categories.category_id WHERE categories.title = "Voyage"
